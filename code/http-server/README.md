# About

- HTTP server example of Express.


# How to create

- basically

```
yarn init
yarn add typescript
yarn tsc --init
yarn add @types/node
```

- add others

```
yarn add express
yarn add @types/express
yarn add ejs
yarn add @types/ejs
```

- Jest

```
yarn add jest # Jest 本体
yarn add @types/jest # Jest の型情報
yarn add ts-jest # Jest 用の TypeScript プリプロセッサ。インストール時に Python が必要です。
```
